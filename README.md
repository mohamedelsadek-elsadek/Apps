# apps — برامج محمد الصادق

مجموعة برامج (HTML، تشتغل أوفلاين) منظّمة بالتصنيف. الصفحة الرئيسية `index.html` هي **مركز الروابط** اللي بيفتح ويشارك كل البرامج.

## التصنيفات والملفات
- **دواجن (poultry/):** layer.html · lab.html · prices.html
- **حياة/حيوانات (livestock/):** cattle-fattening.html
- **أدوية/صيدلية (pharma/):** capa.html
- **مخازن (warehouses/):** trade.html
- **فواتير (invoices/):** genovet-invoices.html · receipt-invoices.html
- **مالية (finance/):** treasury.html · amanat.html · instapay.html
- **شخصي (personal/):** money.html · mona-engagement.html · maintenance.html · family-tree.html · places.html

## الرفع على GitHub Pages
1. اعمل ريبو جديد اسمه **apps**.
2. ارفع كل محتويات المجلد ده (بالمجلدات) مرة واحدة.
3. Settings → Pages → Branch: main → Save.
4. افتح: `https://<username>.github.io/apps/`

## تعديل أي برنامج بعدين
عدّل الملف عندك وارفعه فوق القديم **بنفس الاسم والمسار** → اللينك ثابت والنسخة بتتحدّث فورًا.

## نقل برنامج بين التصنيفات
مش لازم تنقل الملف في GitHub (ده بيغيّر اللينك). غيّر التصنيف من **مركز الروابط** (تعديل الكارت) → اللينك يفضل ثابت.
